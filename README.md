## Описание проекта

Консольное приложение для управления библиотекой, демонстрирующее основные принципы объектно-ориентированного программирования.

## Реализованная функциональность

- Добавление книг в библиотеку
- Выдача и возврат книг
- Поиск книг по автору
- Просмотр доступных книг
- Статистика библиотеки

## ООП-компоненты

1. **Book** — модель книги (Id, Title, Author, Year, IsAvailable; методы CheckOut, Return, ToString)
2. **Library** — управление коллекцией книг (добавление, поиск, фильтрация)
3. **LibraryService** — бизнес-логика (выдача/возврат, статистика, поиск)
